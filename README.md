# me

박연호의 자기소개 페이지입니다.

생각을 구조화하고, 사람에 맞게 전달하며, 끝까지 실행하는 사람

철학과 2학년 박연호입니다.
저는 교내와 교외 활동을 통해 ‘이해 → 전달 → 실행’으로 이어지는 과정을 실제 경험으로 쌓아왔습니다.

교내에서는 철학과 글쓰기와 학생회 활동을 통해 역량을 확장했습니다. 글쓰기를 통해 추상적인 개념을 타인이 이해할 수 있는 흐름으로 풀어내는 연습을 했고, 이를 발표로 공유하며 전달력을 높였습니다. 이후 학생회에서는 엠티, 신입생 환영회, 짝선배-짝후배 프로그램을 기획하고 운영하며 아이디어를 실제로 구현하는 경험을 했습니다. 특히 현장에서 흐름이 어긋났을 때 순서와 방식을 조정하며, 기획은 실행 속에서 완성된다는 것을 체감했습니다.

교외에서는 학원에서 1년 5개월 동안 조교로 근무하며 커뮤니케이션 능력을 길렀습니다. 학생마다 이해 방식이 다르다는 점을 경험하며, 설명을 단순화하거나 과정을 나누고 다양한 예시를 활용하는 방식으로 대응해왔습니다. 이를 통해 전달은 내용뿐 아니라 방식에 달려 있다는 것을 깨달았습니다.

이러한 경험을 바탕으로, 저는 이해한 것을 구조화하고 이를 사람에 맞게 전달하며, 실제로 작동시키는 사람으로 성장해왔습니다.
