# 박연호를 소개합니다.
사고를 정제하고, 사람에 맞게 전달하며, 끝까지 실행하는 힘이 있는 사람, 철학과 2학년 박연호입니다.
철학적 글쓰기를 통해 추상적인 개념을 타인이 이해할 수 있는 언어로 풀어내는 연습을 했고, 이를 발표로 확장하며 전달력을 구체화했습니다. 예를 들어 ‘정의’와 같은 개념을 설명할 때, 단순히 이론을 나열하는 것이 아니라 '결과의 공정성'에 초점을 둔 입장과 '절차의 정당성'에 초점을 둔 입장으로 나누어 설명했을 때 상대의 이해도가 훨씬 높아졌습니다. 
또한 학원 조교로 1년 5개월간 근무하며 전달 방식의 중요성을 더욱 깊이 이해하게 되었습니다. 학생마다 이해 방식과 속도가 다르다는 점을 고려해 개념 설명을 단순화하거나 과정을 나누고, 다양한 예시를 활용하는 방식으로 접근했습니다. 이를 통해 전달은 내용의 정확성뿐 아니라 상대 수준에 맞는 방식의 선택에 달려 있다는 것을 깨달았습니다.
이러한 경험은 학생회 활동으로 이어졌습니다. 엠티, 신입생 환영회, 짝선배-짝후배 프로그램을 기획하고 MC를 맡는 등 직접 운영하며 아이디어를 실제 결과로 구현했습니다. 행사 준비 과정에서 발생할 수 있는 사고들과 발생할 반응들을 사전에 고려하고, 상황에 따라 유연하게 대응하였습니다. 실제로 이번 MT때 레크레이션의 진행을 하며 프로그램의 재미를 위해 사람들의 반응을 살피고 게임을 그 자리에서 수정하는 경험을 하며 기획은 실행 속에서 완성된다는 점을 체감했습니다. 이를 통해 계획을 세우는 것뿐 아니라 끝까지 실행해내는 과정의 중요성을 배웠습니다.
이처럼 저는 사고를 명확히 정리하고 상황에 맞게 전달하며, 이를 실행 가능한 형태로 구현해 실제 결과를 만들어내는 역할을 수행하고자 합니다.
