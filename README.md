# me

박연호의 자기소개 페이지입니다.

제 활동은 크게 교내 활동과 교외 활동으로 나눠볼 수 있습니다.

교내 활동: 생각을 구조화하고, 기획을 현실로 옮기다
철학과에서의 글쓰기는 단순한 과제가 아니었다. 추상적인 개념을 이해하는 데서 그치지 않고, 그것을 다른 사람이 이해할 수 있는 흐름으로 풀어내는 과정이었다. 교내 글쓰기 센터에서 칼럼을 배우고 직접 써보며 생각을 논리적으로 정리하는 법을 익혔고, 이를 글과 발표를 통해 공유해왔다.
이 경험은 학생회 활동으로 확장되었다.
엠티, 신입생 환영회, 짝선배-짝후배 프로그램을 기획하고 운영하며, 행사는 아이디어보다 실행이 중요하다는 것을 체감했다. 준비 자료 제작, 팀 구성, 역할 분배뿐 아니라 현장에서 흐름이 어긋날 때 즉각적으로 수정하는 경험까지 쌓았다. 특히 엠티 레크레이션 진행 중 순서와 방식을 조정해 분위기를 살리며, 기획은 현장에서 완성된다는 것을 배웠다.

교외 활동: 사람에 맞게 전달하는 능력을 익히다
학원에서 1년 5개월 동안 조교로 근무하며 ‘전달’의 방식을 배웠다.
같은 내용도 학생마다 이해 방식이 달랐기 때문에, 설명을 단순화하거나 과정을 나누고, 다른 예시를 활용하는 식으로 방식을 조정했다.
이 과정을 통해 전달은 내용보다 방식에 달려 있다는 것을 깨달았고, 다양한 학생들을 상대하며 상황에 맞게 대응하는 실전 커뮤니케이션 능력을 갖추게 되었다.
