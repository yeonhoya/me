# me

박연호의 자기소개 페이지입니다.

사고를 정제하고, 사람에 맞게 전달하며, 끝까지 실행하는 힘이 있는 사람, 철학과 2학년 박연호입니다. 저는 ‘사고 → 전달 → 실행’으로 이어지는 과정을 하나의 역량으로 삼아 실제 경험 속에서 발전시켜왔습니다.

철학과 글쓰기를 통해 추상적인 개념을 타인이 이해할 수 있는 흐름으로 풀어내는 연습을 했고, 발표를 통해 전달력을 구체화했습니다. 단순한 이해를 넘어, 상대의 관점에서 내용을 재구성하는 과정의 중요성을 깨달았습니다. 이러한 경험은 학생회 활동으로 이어졌습니다. 엠티와 신입생 환영회, 짝선배-짝후배 프로그램을 기획하고 운영하며 아이디어를 실제 결과로 구현했습니다. 특히 예상과 다른 상황에서도 현장의 흐름에 맞게 순서와 방식을 조정하며, 기획은 실행 속에서 완성된다는 점을 체감했습니다.

또한 학원 조교로 1년 5개월간 근무하며 전달 방식의 중요성을 배웠습니다. 학생마다 이해 방식이 다르다는 점을 고려해 설명을 단순화하고 과정을 나누는 방식으로 접근했습니다. 이를 통해 전달은 내용뿐 아니라 방식에 달려 있음을 깨달았습니다.

이처럼 저는 이해한 것을 구조화하고, 상황과 대상에 맞게 전달하며, 실제로 작동하는 결과로 만들어내는 사람으로 성장해왔습니다.
