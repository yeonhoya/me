# me

박연호의 자기소개 페이지입니다.

생각을 구조화해 전달하고, 그것을 실제로 작동시키는 사람, 철학과 2학년 박연호입니다. 저는 윤리학, 특히 AI에 대해 내리는 판단의 기준에 관심이 있습니다. 같은 상황에서도 서로 다른 판단이 나오는 이유가 단순한 오해가 아니라 각자가 전제한 윤리적 기준의 차이에서 비롯된다는 점이 흥미로웠고, 이는 AI에 대한 판단 기준은 어떻게 설정되어야 하는가라는 질문으로 이어졌습니다.

이 질문을 탐구하기 위해 뉴스 기사와 SNS 게시글을 데이터로 수집하고, 자연어 처리 기반 AI를 활용해 텍스트 속 판단의 근거를 추출한 뒤 공리주의와 의무론 등의 윤리 이론에 따라 분석하고자 합니다. 이를 통해 실제 사회의 판단이 어떤 윤리적 구조 위에서 작동하는지 밝히고, AI 판단 기준 설계에 시사점을 도출하고자 합니다.

철학과 글쓰기, 학생회 활동, 학원 조교 경험을 통해 복잡한 개념을 구조화해 전달하는 능력을 길러왔습니다. 5년 후에는 기술과 윤리의 접점에서 판단의 구조를 분석하고 기준을 설계할 수 있는 연구자로 성장하고자 합니다.
